About the Programme

What is Ryan Tech Skills Camp?

Ryan Tech Skills Camp is an intensive 8-week holiday programme by Ryan Tech Hub — De Programmers Community. Built specifically for JSS3 students, it gives young people real, practical digital skills before Senior Secondary begins.

This is not a classroom lesson repeated. It is a transformation — designed for the world your child is actually stepping into.

Curriculum

What Your Child Will Learn

Four skill areas — each chosen because it matters right now, not in theory, not someday. Right now.

🤖
AI Basics
Understanding and using the tools reshaping the world right now. Your child will learn how AI works, how to use it productively, and how to think critically about it — not just consume it.
🌐
Web Development
How to build real things on the internet. From writing a first line of code to seeing a webpage come to life — this is where ideas stop being ideas.
💻
Computer Skills
The foundation every student needs before Senior Secondary. File management, productivity tools, typing efficiency, and digital workspace organisation.
📱
Digital Literacy
How to think, navigate, and operate in a digital world. Online safety, information evaluation, and using technology with purpose rather than just scrolling.
